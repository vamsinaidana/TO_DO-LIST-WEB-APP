📝 Modern Todo List App

A responsive and modern Todo List web application built using HTML, CSS, JavaScript, and Bootstrap with a sleek glassmorphism UI.

🌟 Features

✅ Add, delete, and mark tasks as completed

📅 Add date & time to tasks

💾 Local storage support (data persists after refresh)

📱 Fully responsive design (mobile-friendly)

🎨 Modern glassmorphism user interface

⚡ Smooth and interactive user experience

🛠️ Technologies Used

HTML5

CSS3 (Glassmorphism design)

JavaScript (DOM manipulation & localStorage)

Bootstrap 5

🎯 Purpose

This project demonstrates front-end development skills by building a clean, functional, and visually appealing task management app.

🚀 Future Improvements

🔔 Notifications & reminders

📊 Task filtering (All / Completed / Pending)

🌙 Dark mode toggle

☁️ Backend integration (Firebase / API)
